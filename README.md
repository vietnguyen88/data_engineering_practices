## Getting Started
### Prerequisites

- Python 3.11 / 3.12 / 3.13
    - ```python --version```

- pip
    - ```pip --version```

- Git

### 1. Clone the repository

```bash
git clone https://github.com/khanhnguyen-08/data_engineering_practices.git

cd data_engineering_practices
```



### 2. Creat virtual environment (optional, but recommended)

**Window**

Create a virtual environment
```bash
python -m venv .venv
```

Activate the environment

```bash
.\.venv\Scripts\activate
```

### 3. Install packages

Using Pip to install packages from requirements.txt file
```bash
pip install -r requirements.txt
```
